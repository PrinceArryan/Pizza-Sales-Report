📌 Project Overview

The Pizza Sales Dashboard is an interactive Business Intelligence solution developed in Power BI to analyze pizza sales performance throughout 2015. The dashboard provides valuable insights into revenue, customer purchasing behavior, product performance, and sales trends, enabling business stakeholders to make informed decisions for improving sales and operational efficiency.

🎯 Objectives
Monitor overall sales performance using key business KPIs.
Identify best and worst selling pizza categories.
Analyze daily and monthly ordering trends.
Understand customer preferences by pizza category and size.
Support data driven decision making through interactive visualizations.

The dashboard summarizes the business performance using the following KPIs:

Total Revenue: 817.86K
Total Orders: 21,349
Total Pizzas Sold: 49,573
Average Order Value: 38.31
Average Pizzas per Order: 2.32

Tools & Technologies
Power BI – Dashboard Development
SQL – Data Extraction & Analysis
Power Query – Data Cleaning & Transformation
DAX – KPI & Measure Creation
Microsoft Excel/CSV – Data Source
